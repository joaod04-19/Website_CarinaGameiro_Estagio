# Website_CarinaGameiro_Estagio
Desenvolvimento de Website para Solicitadora.

# Descrição
Este projeto consiste no desenvolvimento de um website institucional para apresentação dos serviços da Solicitadora Carina Gameiro.
O website foi desenvolvido no âmbito de um estágio curricular e tem como objetivo disponibilizar informação clara sobre os serviços prestados, facilitar o contacto com clientes e melhorar a presença online.


# Tecnologias Utilizadas
. HTML;

. CSS;

. JavaScript;


# Estrutura do Website
O website é composto pelas seguintes páginas:

. Home (index.html) – página principal do website;

. Sobre (sobre.html) – apresentação da profissional;

. Serviços (servicos.html) – descrição dos serviços disponíveis;

. Atendimento (atendimento.html) – informações sobre atendimento e contacto;

. Testemunhos (testemunhos.html) – testemunhos de clientes ao serviço prestado;


O principal objetivo deste projeto foi desenvolver um website funcional, organizado e visualmente apelativo, aplicando conhecimentos de desenvolvimento web adquiridos durante a formação.

Este projeto foi desenvolvido por João Doroteia no âmbito de estágio curricular.
